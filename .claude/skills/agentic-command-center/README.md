# agentic-command-center

A guided planning skill for architecting an AI command center for a business: a single AI interface that understands a request, routes it to the right internal system, and acts.

Source: [Forgemind-git/agentic-command-center](https://github.com/Forgemind-git/agentic-command-center)

## Phases

1. **Own your systems** — map business operations into internal dashboards you control
2. **Connect your tools** — integrate existing services (WhatsApp, email, calendars, etc.)
3. **Expose them over MCP** — create safe, AI-callable tool interfaces
4. **Unify under one command layer** — deploy a single AI coordinator

## Usage

Run `/agentic-command-center` in Claude Code and describe your business — it walks through the four phases above, one at a time.
